# Sobre o Projeto

Este projeto é um trabalho acadêmico baseado no desafio da RSNA , onde o objeto é o treinamento de um modelo inteligente para auxiliar na detecção de possíveis tumores em imagens de mamografias.
Link do Desafio: [RSNA Screening Mammography Breast Cancer Detection](https://www.kaggle.com/competitions/rsna-breast-cancer-detection/overview)

## Ferramentas utilizadas
-Python3
-Google Colab
-YOLOv8
